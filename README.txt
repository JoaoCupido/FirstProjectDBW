Como instalar e iniciar o servidor:
1. No terminal, correr o comando 'npm install' para instalar as bibliotecas necessárias.
2. Atualizar as bibliotecas usando o comando 'npm update'.
3. Depois, executar 'node server.js' para iniciar o servidor.
4. Abrir o seu browser de escolha e ir para 'localhost:3000'.
